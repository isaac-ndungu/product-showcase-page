# Toteme Product Showcase Page
A minimilist fashion-focused product showcase page inspired by a modern fashion brand.

## Contributor
- Isaac Ndungu

## Project Brief 
This project is a static e-commerve product showcase page that simulates a real world online shopping experience.

## Technology Used 
- HTML5
- CSS
- Font Awesome
- Google Fonts

## Core Features 
- Product cards with prices product title and price
- A static shopping cart
- Responsive design that adapts from the desktop layout to mobile viewports
